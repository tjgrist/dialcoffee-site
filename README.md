# dialcoffee.site

Static marketing + legal site for **Dial Coffee** (iOS app), served via
GitHub Pages at <https://dialcoffee.site>.

This repo is a **publish target only** — the source of truth is the `site/`
directory in the private app repo. Do not edit files here by hand; they are
overwritten on the next publish. To update, run `tools/publish-site.sh` from
the app repo.

Pages: deployed from the `main` branch root. Custom domain in `CNAME`.
